# Brand Guideline Color Usage

## Color Variables Mapping

### Light Theme
- `--primary`: #F4D03F (yellow) - Buttons, hero sections, section titles, major UI
- `--accent-1`: #991B1B (crimson) - Links, emphasis, quote blocks
- `--accent-2`: #00D9A3 (green) - Secondary accents, hover states

### Dark Theme  
- `--primary`: #4A7FDB (blue) - Buttons, hero sections, section titles, major UI
- `--accent-1`: #86EFAC (electric green) - Links, emphasis, quote blocks
- `--accent-2`: #F4D03F (yellow) - Secondary accents, hover states

## Component Usage Patterns

### Buttons
- Background: `var(--primary)`
- Text color: `var(--border)` (black in light, white in dark)
- Hover: Can use `var(--accent-2)`

### Links  
- Color: `var(--accent-1)`
- Underline decoration: `var(--accent-1)`

### Hero Sections
- Background: `var(--primary)`
- Text: `var(--border)`

### Section Titles
- Background: `var(--primary)`
- Text: `var(--border)`

### Quote Blocks
- Background: `var(--accent-1)`

### List Item Borders
- Border-left: `var(--primary)`

### Strong/Bold Text
- Color: Can use `var(--accent-1)` for emphasis

### Secondary Elements
- Use `var(--accent-2)` for hover states and decorative elements
