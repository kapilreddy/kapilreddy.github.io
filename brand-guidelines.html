Body text maintains excellent readability with high contrast ratios. We use near-black on off-white for light theme and off-white on rich black for dark theme. Links are bold and prominently underlined to ensure they stand out from regular text. The Inter font family provides optimal legibility at all sizes.

Secondary text uses a slightly smaller size and muted color for hierarchy. Perfect for captions, metadata, and supporting information that shouldn't compete with primary content.

CODE EXAMPLES

// Neobrutalist theme configuration
const brutalistTheme = {
  primary: '#F4D03F',
  accent: '#D94A6A',
  border: '3px solid black',
  shadow: (offset) => `${offset}px ${offset}px 0 black`
};

function applyBrutalStyle(element) {
  element.style.border = brutalistTheme.border;
  element.style.boxShadow = brutalistTheme.shadow(8);
  return element;
}
/* Brutalist button styles */
.btn-brutal {
  background-color: var(--primary);

NEOBRUTALIST PRINCIPLES

  • Use minimum 3px borders on all interactive elements
  • Apply 8-12px offset shadows for depth and dimension
  • Embrace asymmetry with intentional misalignment
  • Stack colored rectangles with visible borders between layers
  • No gradients - only flat, solid colors
  • Use only rectangles and perfect circles - no rounded corners
  • Let colored blocks overflow their containers for dynamic layouts
  • Incorporate ASCII decorations (▰▱▰ ◐◑◒◓) as design elements
  • Rotate elements slightly (-3deg to 3deg) for organic feel
  • Use monospace fonts for headers and UI elements
