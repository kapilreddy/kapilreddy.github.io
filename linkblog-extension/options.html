Linkblog Capture Settings

Specify a relative path within your Downloads folder (e.g., "linkblog-files/"). Leave empty to save directly to Downloads. The path will be created if it doesn't exist.

When enabled, files will be saved automatically to the default location without showing the "Save As" dialog.

Tips

  • The path is relative to your Downloads folder
  • Use forward slashes (/) in the path
  • End the path with a slash (/) for directories
  • Examples: "blog/links/", "markdown-files/", or leave empty
